# The Farmer Was Replaced - Saves

[The Farmer Was Replaced](https://store.steampowered.com/app/2060160/The_Farmer_Was_Replaced/) is a game on Steam, with the following description:

    Program and optimize a drone to automate a farm and watch it do the work for you. Collect resources to unlock better technology and become the most efficient farmer in the world. Improve your problem solving and coding skills.

## Python

The game lets you write Python code by using there known methods, like:

- harvest()
- move(..) > North, East, South, West
- plant(..) > Entities.Grass, Entities.Tree, Entities.Carrot ..
- ...

## External Editor - VS Code

There is a possibility to program in-game. Another possibility to program is to use an external editor. In my case VS Code. Once I switched to an External Editor, ofcourse it had to need version control, therefor this repository exists.
